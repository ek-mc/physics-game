# Physics Game (GR)

Status: Active | Last release: v0.8.19 | Last update: 2026-03-15 | Live: https://ek-mc.github.io/physics-game/


Μικρό quiz app φυσικής στα ελληνικά με ερωτήσεις πολλαπλής επιλογής (ABCD) και σκόπιμες παγίδες.

## Features
- Ενότητες: Κινηματική, Δυναμική, Περιστροφή, Έργο-Ενέργεια, Ταλαντώσεις
- Μεγάλη τράπεζα ερωτήσεων (500+ generated questions)
- Επίπεδα δυσκολίας: Easy / Medium / Hard / Μικτή
- Πλήθος ερωτήσεων ανά γύρο: 10 / 20 / 30
- Timer mode με επιλογή χρόνου ανά ερώτηση (20/30/45/60s)
- Streak mode (τέλος στο πρώτο λάθος)
- Best score + best streak (global, αποθήκευση localStorage)
- Αποθήκευση επιλογών menu (difficulty/count/timer/streak/time-per-question) σε localStorage
- Επιπλέον ερωτήσεις ερμηνείας γραφημάτων (x-t, v-t)
- Νέα exam-style packs σε Έργο-Ενέργεια (sand penetration, spring/friction mixes, F-x area)
- Ενισχυμένο Definitions/Notation section (W, K, U, P, Joule/Watt, conservative force traps)
- Άμεσο feedback με σωστή μέθοδο/τύπο
- Απλό static app (HTML/CSS/JS)

## Run
Άνοιξε το `index.html` σε browser.
